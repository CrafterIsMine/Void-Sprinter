# Void-Sprinter
A game about sprinting in the void that allows the player to collect points in outer space while running from enemies and dashing into them to achieve scores. The game is made with HTML, CSS, JS and canvas and features a trippy hexagonal background to distract the player while providing the player with the challenge of never ending waves of enemies. 

**Controls:**
Player can use WASD for movement and press space to dash through enemies or for increasing movement speed and collecting points in the meantime. The goal is to achieve as much score as possible.

**Inspiration :** I felt there are not enough games with mechanics like this being made in this year and time, so i decided to make it myself and the unique part about this game is the movement of the player that goes really smooth with the dash combat.

**Theme:** Indie Gamedev

The game fits the theme because its unique from other games and really simple with not a massive budget but a solo dev project like this and unique in the sense that you can actually defend yourself and have freedom to travel anywhere in the map and it brings back the core memory of old arcade games to the player playing it.
